# Icon pro extension

Pro funkční extension potřebujete vytvořit ikonku `command-icon.png` v root složce.

## Požadavky na ikonku:

- **Formát**: PNG
- **Rozměry**: 512x512 px (doporučeno)
- **Styl**: Jednoduchý, čitelný i v malé velikosti
- **Barva**: Preferovaně jednobarevná nebo s jednoduchým gradientem

## Možnosti vytvoření:

1. **Použít SF Symbol** (nejjednodušší):
   - Najděte vhodný symbol na https://developer.apple.com/sf-symbols/
   - Exportujte jako PNG 512x512

2. **Vytvořit vlastní**:
   - Použijte Figma, Sketch, nebo Photoshop
   - Export jako PNG 512x512

3. **Použít placeholder**:
   - Pro testování můžete použít jakýkoliv 512x512 PNG obrázek

## Doporučené symboly pro ComfyUI extension:

- `wand.and.stars` (magická hůlka)
- `photo.stack` (stack fotek)
- `rectangle.stack.fill` (stack obdélníků)
- `sparkles` (jiskry)
- `slider.horizontal.3` (posuvníky)

Po vytvoření ikonky ji přejmenujte na `command-icon.png` a umístěte do root složky projektu.
